# Nexus App Build

## Dossier pour les icônes de l'application

Placez vos icônes ici:
- `icon.icns` - Icône macOS (512x512px minimum)
- `icon.ico` - Icône Windows (256x256px minimum)

Si vous n'avez pas d'icônes, electron-builder utilisera une icône par défaut.

### Générer les icônes

Vous pouvez utiliser des outils en ligne pour convertir une image PNG:
- https://cloudconvert.com/png-to-icns (pour macOS)
- https://cloudconvert.com/png-to-ico (pour Windows)
